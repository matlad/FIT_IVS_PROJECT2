# Kalkulacka

Jednoduchý nastroj umožnující jednoduché matematické operace jako
* sčítání
* odčítání
* dělení
* násobení
* obesnou mocninu
* obecnou odmocnimnu
* faktorial
* modulo (zbytek po dělění)

na množině complexních čísel.
